# Throw 断舍离助手

一个帮你理性断舍离的轻量 Web App。

## 功能

- **物品录入**：记录物品名称、类别、价格、品相、上次使用时间
- **智能评估**：根据多维度自动评分，给出明确的行动建议（保留 / 冷静期 / 处理掉）
- **差异化建议**：根据物品价格和品相推荐最合适的处理方式（二手出售 / 送人 / 捐赠 / 直接丢弃）
- **冷静室**：纠结的物品先放一放，到期自动提醒
- **处理记录**：记录每件物品的最终去向
- **数据统计**：可视化你的断舍离进度

## 使用

纯前端，零依赖。直接打开 `declutter.html` 即可使用，数据存储在浏览器 localStorage 中。

## 技术栈

- 纯 HTML / CSS / JavaScript
- Zen Maru Gothic 字体
- 无框架依赖，单文件部署
